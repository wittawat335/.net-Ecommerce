# ASP.NET CORE MVC + ASP.NET CORE WEBAPI
Web application เป็นส่วนในการจัดการข้อมูลหลังบ้านเช่น รายการสินค้า บัญชีผู้ใช้งาน ตำแหน่งผู้ใช้งาน เมนู สิทธิ์การใช้งานเมนูตามตำแหน่ง
## Tech Stack

**Client:** Html, CSS, JavaScript 

**Server:** C#

# Database Tool
- Dapper
- SQL Sever 2019
- Store Procedure
